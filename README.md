# ИНСТРУКЦИЯ ПОЛЬЗОВАТЕЛЯ

## 1. Введение
Данная система (Leave Management System) предназначена для автоматизации процессов подачи и обработки заявок на отпуска сотрудников. Целевая аудитория — сотрудники компании, планирующие отпуск, а также HR-менеджеры и руководители, ответственные за утверждение заявок.

## 2. Требования
- **Устройство:** Персональный компьютер, ноутбук или планшет.
- **Браузер:** Современные веб-браузеры (Google Chrome, Mozilla Firefox, Safari, Edge).
- **Условия работы:** Наличие стабильного подключения к корпоративной сети или интернету.

## 3. Установка и запуск
1. Клонируйте репозиторий проекта на локальный компьютер.
2. Перейдите в директорию проекта.
3. Установите модули из `requirements.txt`.
4. Запустите через `uvicorn main:app --reload`.

## 4. Основные функции

### 4.1 Регистрация
Для начала работы сотруднику необходимо создать личный профиль, указав имя, должность и корпоративный email, после чего нажать кнопку «Зарегистрироваться».
![Регистрация](https://raw.githubusercontent.com/almaskenjegarin-code/django-project/main/static/img/screenshot_registration.png)

### 4.2 Авторизация
После регистрации пользователь вводит свои учетные данные (логин/email и пароль) в окно входа для доступа в систему.
![Авторизация](https://raw.githubusercontent.com/almaskenjegarin-code/django-project/main/static/img/screenshot_login.png)

### 4.3 Основной функционал
Ключевая функция проекта — возможность быстро создать заявку на отпуск (указав даты отдыха и тип отгула), а также отслеживание статуса заявки (на рассмотрении, одобрена, отклонена). Административная панель позволяет руководству управлять всеми запросами централизованно.
![Основной функционал](https://raw.githubusercontent.com/almaskenjegarin-code/django-project/main/static/img/screenshot_home.png)

## 5. Работа с системой (пошагово)
- **Шаг 1:** Войдите в систему под своей учетной записью.
- **Шаг 2:** В главном меню выберите раздел «Создать заявку» (Create leave requests).
- **Шаг 3:** Заполните форму: выберите дату начала, дату окончания и причину отпуска, затем нажмите «Отправить заявку».
- **Шаг 4:** Проверяйте статус вашей заявки в личном кабинете, чтобы узнать решение руководителя.
![Работа с системой](https://raw.githubusercontent.com/almaskenjegarin-code/django-project/main/static/img/screenshot_step.png)

## 6. Частые ошибки
- **Ошибка:** Не удается войти в систему ("Неверный логин или пароль").
  - **Решение:** Проверьте правильность вводимых данных или обратитесь к администратору для сброса пароля.
- **Ошибка:** Сервер не отвечает после запуска.
  - **Решение:** Убедитесь, что установлены все библиотеки из `requirements.txt`, а базы данных PostgreSQL и Redis запущены и корректно настроены.

## 7. Заключение
Использование Leave Management System делает процесс оформления отпусков прозрачным, значительно экономит время сотрудников и снижает бумажный документооборот внутри компании.
